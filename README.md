# audio_transcriber
Transcodes audio files to text, supports MP3, M4A, WAV. Does File Conversion. No Online API's. Python 3

Install prequisites

To run the app you need several things installed:

    Python 3
    the module pydub
    the module optparse
    the module filetype
    the module tqdm
    the program ffmpeg
    the module SpeechRecognition
